FAQ

মাথায় কিছু প্রশ্ন?

Download বা install-এর আগে যা জানা দরকার — সোজাসাপ্টা উত্তর।

কেন শুধু ZIP download?

Website থেকে real folder direct download করা যায় না। ZIP হলো browser-safe way, আর extract করলে একটাই TechSubxBD folder পাওয়া যায়।

ZIP extract করার পর যে TechSubxBD folder তৈরি হবে, সেটাই Load unpacked-এ select করবেন।

এই website থেকে latest ZIP download করে আগের folder replace করুন, তারপর Chrome extension page থেকে reload করুন।
